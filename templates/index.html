Prédection de la qualité d'un vin sur la base d'éléments physico-chimiques.

Pour obtenir le score de qualité d'un vin on a besoin d'une entrée .json avec les informations suivantes pour le vin à noter:

  • fixed acidity
  • volatile acidity
  • citric acid
  • residual sugar
  • chlorides
  • free sulfur dioxide
  • total sulfur dioxide
  • density
  • pH
  • sulphates
  • alcohol
  • quality
Interrogez l'api sur le endpoint /predict
Ce point d'accès accepte la méthode POST avec des données d'entrée JSON et doit retourner les prédictions. Il suffit d'envoyer une requête POST sur le endpoint /predict de type :
requests.post(/predict, json=input_simple)

response = requests.post("https://wineprojectflask.herokuapp.com/predict", json={ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] })

print(response.json())

ou une requête curl du type
$ curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://wineprojectflask.herokuapp.com/predict'
